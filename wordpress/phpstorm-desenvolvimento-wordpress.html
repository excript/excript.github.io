PHPStorm

A IDE PHPStorm é a ferramenta mais utilizada no desenvolvimento com a linguagem PHP. Com a mesma, conseguimos facilmente criar novos arquivos HTML, JS, PHP e também, conseguimos facilmente depurar scripts, tanto PHP como também, scripts JS.

Existem versões disponíveis para Windows, Linux e Mac.

Obviamente, para programar em PHP, não é necessário a utilização de IDE, porém, quando fazemos uso de uam, conseguimos evitar uma série de erros e também, agilizar tarefas rotineiras.
