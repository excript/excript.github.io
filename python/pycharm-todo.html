GERENCIANDO MARCAÇÕES TODO COM A IDE PYCHARM

Publicado em 02-10-2017 por Cláudio Rogério Carvalho Filho
  • ARTIGO

Neste artigo aprenderemos como adicionar marcações dentro do nosso código e gerenciar essas marcações pela View ToDo do PyCharm

VIEW TODO DO PyCharm

O PyCharm possui um excelente recurso para organizar nosso trabalho pela adição de uma palavra chave em comentários e a descrição do que devemos fazer. Este recurso ainda que simples de ser utilizado causa bastante dúvidas e problemas pelo fato de que muitos tentam inserir tarefas dentro de projetos gigantes e o PyCharm não sabe por qual projeto se orientar.

Portanto, para que possamos testar o recurso de adicionarmos marcações dentro do nosso código e esses itens irem sendo adicionados na lista ToDo, façamos o seguinte.

  1. Inicialmente, crie um novo diretório
  2. Criado o diretório, vamos no PyCharm em File -> New Project e em seguida, vamos até o diretório que acavamos de criar
  3. Após o projeto criado, vamos adicionar um arquivo de nome main.py
  4. No arquivo main.py vamos adicionar as seguintes linhas de código:
#todo: o que fazer 1
#todo: o que fazer 2

print("testando o recurso ToDo")

#todo: o que fazer 3
#todo: o que fazer 4

Feito isso, pressione as teclas Alt + 6 ou vá no menu View -> TODO. Feito isso, será aberta uma View na parte inferior do PyCharm, semelhante ao que temos na imagem a seguir:

PERSONALIZANDO A MARCAÇÃO

Estudamos a ferramenta principal para o gerenciamento de marcações que adicionamos em nosso código, porém, podemos ir muito mais longe adicionando marcações personalizadas e até, definindo diferentes estilizações.

Por exemplo, podemos definir a marcação "estudo:", logo, sempre que essa palavra for reconhecida dentro dos arquivos do nosso projeto, a linha será adicionado à View TODO do PyCharm.

Vamos adicionar a marcação estudo. Pra isso, vamos abrir a telas de configurações do PyCharm. Pressione as tecals Ctrl + Alt + S ou então, vá no menu File -> Settings. Feito isso, a tela contendo todas as configurações do PyCharm será a aberta. O primeiro campo permite que façamos uma busca dentro das várias seções de configuração, vamos digitar a palavra "todo".

Feito isso, a sua tela deverá se parecer com a imagem que temos abaixo:
