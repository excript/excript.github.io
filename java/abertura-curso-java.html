INTRODUÇÃO

Seja muito bem-vindo ao Curso de Java da eXcript. É com muita satisfação que tornamos públicos um material onde qualquer pessoa consiga aprender os primeiros passos para se tornar um programador!

Nesse primeiro momento, vamos estudar uma simples demonstração da linguagem Java. Por isso, não se preocupe com absolutamente nada, até porque, nessa primeira vídeo aula temos um demonstração para que você possa ter um primeiro contato com a linguagem.

O exemplo que realizamos nessa primeira aula, o famoso Hello World, será explicado em detalhes na sequência! Como já dito, nosso principal objetivo foi demonstrar um pouco do Java e também, provar que é fácil e qualquer pessoa consegue aprender a programar!

Caso tenhas curiosidade, abaixo está o código na integra que escrevemos na primeira aula.

EXEMPLO FEITO EM AULA

package com.excript.helloworld;

public class HelloWorld{
    public void static main(String[] args){
        System.out.println("Hello World.");
    }
}

De fato, a única linha "importante" nesse código é a linha a seguir:

System.out.println("Hello World");

O código acima, envia para o console a frase "Hello World" e para isso, utilizamos a função de nome println que está contida no pacote System. O nome println significa, print line, que numa tradução livre do Inglês, seria: Imprima uma linha.

Então, qualquer texto que for colocado entre as aspas, é enviado para o Console, ou seja, será impresso na tela.
