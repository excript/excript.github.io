ECLIPSE

O programa Eclipse é uma ferramenta para auxiliar o desenvolvimento de programas. Podemos utiliza-lo para programarmos em várias linguagens, como por exemplo Java, PHP, JavaScript, C, C++ e etc.

Para desenvolver programas em Java, podemos utilizar qualquer editor de texto básico, como por exemplo, o Notepad do Windows. Isso porque, o único programa que realmente é necessário no desenvolvimento é a JVM que é instalada junto com o JDK e utilizada para executar os programas escritos em Java.

Então, podemos criar arquivos com extensão *.java e em seguida ir no prompt de comando, compilar e em seguida, executar o código gerado. Então, é super importante que saibamos que a utilização da ferramenta Eclipse é opcional, e a finalidade de utilizarmos a mesma é para tornar o nosso trabalho de codificação mais simples e rápido.

É importante entender que o Eclipse é um programa, ou melhor, um editor de texto e pode ser substituido por qualquer outro editor.

O Eclipse é um software escrito em Java, gratuito, possui o seu código-fonte disponibilizado e é desenvolvido pela organização sem fins lucrativos Eclipse Foundation.

Devido ao fato do Eclipse ser totalmente desenvolvido em Java, o mesmo pode ser executado em qualquer dispositivo que tenha instalado a JVM.

O Eclipse não possui um instalador, então, iremos baixar um arquivo compactado, que conterá um executável e também, todas as bibliotecas necessárias para a sua execução e também, um executável.

INSTALANDO O ECLIPSE

Antes de instalarmos o Eclipse, é necessário que tenhamos instalado o Java em nosso computador, do contrário o Eclipse não funcionará, até porque, o mesmo foi desenvolvido em Java.

PASSO-A-PASSO

  1. Ir no site da Oracle e baixar o http://www.oracle.com/technetwork/pt/java/javase/downloads/index.html[SDK]
  2. Instalar o SDK
  3. Ir no site do de download do Eclipse e baixar a instalação para Windows ou Mac
  4. Descompactar o arquivo baixado
  5. Entrar na pasta Eclipse e executar o Eclipse

Note

O Eclipse não possui um instalador! Basta somente fazer o download, descompactar e começar a trabalhar!

MENSAGENS DE ERROS NA EXECUÇÃO DO ECLIPSE

Não é comum a ocorrência de problemas na execução do Eclipse, até porque, o mesmo é desenvolvido em Java e assim, quem executa o programa é a máquina virtual e não o sistema operacional. Porém, alguns computadores possuem mais de uma instalação da máquina virtual do Java e assim, pode acontecer de que ao tentarmos executar o Eclipse, apareça alguma mensagem de erro.

A maior parte dos problemas que ocorrem quando estamos começando a programar em Java, se deve ao fato de que o nosso computador possui versões antigas da JVM. Caso você execute o Eclipse e ao invés de abrir o programa seja exibido alguma mensagem de erro, a primeira coisa a ser feita para tentar resolver o problema é, averiguar se existem versões antigas da JVM instalada. Se houver, desinsta-le a(s) versão(ões) antiga(s) e também, a versão da JVM (Java SE) que você instalou recentemente.

Em seguida, baixe a última versão do Java SE e reinsta-le-a.

CASO OS PROBLEMAS PERSISTAM

Se por alguma razão você não conseguir executar o Eclipse, a melhor opção é desinstalar tudo que for referente ao Java. Feito isso, baixe a última versão do JDK e instale tudo novamente.

Após a desinstalação, vá até a pasta Program Files, Arquivos de Programas e certifique-se de que a pasta do Java foi excluída. Entre nos 2 diretórios a seguir e certifique-se de que a pasta Java não exista ou então, que a mesma esta vazia.

  • C:\Program FileszJava
  • C:\Program Files (x86)zJava

Caso exista alguma pasta dentro de um dos diretórios acima, não delete-os !! Vá no desinstalador de programas do Windows e então, faça a desinstalação da maneira correta.

Para acessar o desinstalador do Windows, acesse o diretório a seguir:

  • Painel de ControleProgramasProgramas e Recursos

Se ainda assim você tiver problemas, coloque a mensagem de erro no Google e veja se outras pessoas estão recebendo a mesma mensagem.

DESINSTALADOR DE PROGRAMAS DO WINDOWS

Na janela de instalação e desinstalação de programas do Windows, você encontrará um lista de todos os programas que estão instalados no seu computador. Procure nessa lista todos os nome de programas que comecem com a palavra "Java" e desinsta-le-os. Por exemplo, você pode encontrar alguns programas com os seguintes nomes ou então, com nomes semelhantes:

  1. Java 7
  2. Java 8
  3. Java SE
  4. Java SDK

Após certificar-se de ter desinstalado todas as versões antigas da JVM, faça o download [caso ainda não tenhas feito] do JDK e instale o mesmo em seu computador.

Em seguida, tente executar novamente o Eclipse e averigue se ainda está ocorrendo algum erro.
